папка KodProject - (Mesa + SimPy + PySD) модель очереди в банке
Реализовано:
Агентная логика (Mesa):
Клиенты двух типов (стойкие и чувствительные к жаре) заходят в банк с заданной вероятностью.
Визуализировано движение клиентов: вход, ожидание в очереди, проход к кассе и выход из здания.
Агенты удаляются из системы после выхода, чтобы не перегружать память.
Процесс обслуживания (SimPy):
Реализовано ограниченное количество окон (кассиров).
Механизм терпения: если время ожидания в очереди превышает порог терпения клиента, он уходит недовольным (помечается красным цветом).
Синхронизация шагов Mesa с дискретными событиями SimPy.
Климатическая динамика (PySD):
Температура в банке рассчитывается через внешнюю модель системной динамики (.mdl).
Обратная связь: чем больше людей в банке, тем быстрее растет температура.
Влияние на поведение: при повышении температуры запас терпения у всех клиентов сокращается пропорционально жаре.
Визуализация (Matplotlib):
Интерактивное окно с анимацией в реальном времени.
Цветовая кодировка: Синий/Фиолетовый (идут/ждут, стойкие к жаре/чувствительные к жаре), оранжевый квадрат (обслуживаются), зеленый (успех), красный (ушел из-за жары).
Динамический заголовок с текущей температурой и статистикой очереди.
